# Venurize

Independent software studio in Bendigo, Victoria. Web platforms, custom applications and autonomous workflows.

The site is a single self-contained static file: [index.html](index.html). No build step, no dependencies.

## Deploy

Any static host works. For Vercel: import this repo as a new project (framework preset: Other), then add `venurize.com` under Domains and follow the DNS instructions.
